# search-books-front
도서 이미지에서 ISBN을 자동으로 추출하고, 도서 정보를 저장하는 시스템이다.

## 프로젝트 개요
search-books 프로젝트의 백엔드와 연동되는 React FrontEnd입니다.

## 기술 스택
- **FrontEnd** : `React 19`, `React Router`, `Axios`
- **Build Tool** : `Create React App`
- **Package Manager** : `npm`
